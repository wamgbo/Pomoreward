# 🍅 Pomoreward

**Pomoreward** 是一個結合「番茄工作法」與「獎勵機制」的生產力工具。專注工作累積「抽獎次數」，換取你想要的自定義獎勵。

---

## ✨ 功能特點

* **專注計時**：標準 25/5 分鐘番茄鐘（支援在 Focus 時切換為紅色背景）。
* **獎勵機制**：完成專注時段即可獲得抽獎機會。
* **幸運轉盤**：透過轉盤抽取獎品，增加專注樂趣。
* **自定義獎勵**：自由設定獎品名稱與內容。
* **我的背包**：存放並管理已獲得的獎勵卡片。
