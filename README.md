🍅 Pomoreward

Pomoreward 是一個結合「番茄工作法」與「獎勵機制」的生產力工具。專注工作累積點數，換取你想要的獎勵。
✨ 功能特點

    專注計時：標準 25/5 分鐘番茄鐘。

    點數累積：完成專注時段即可獲得點數。

    獎勵兌換：自定義獎勵清單，消耗點數兌換小確幸。

    簡潔介面：極簡設計，拒絕分心。
